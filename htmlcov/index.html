Pure3270 Code Coverage Reports

Note: This page provides access to automatically generated code coverage reports. These reports are updated with each commit to the main branch.

Available Reports

  • Coverage Report (Main) - Detailed coverage information

Local Report Generation

You can also generate these reports locally:

# Install dependencies
pip install .[test]

# Generate coverage report
pytest --cov=pure3270 --cov-report=html

# Open the report in your browser
# open htmlcov/index.html (macOS)
# xdg-open htmlcov/index.html (Linux)
# start htmlcov/index.html (Windows)

Linting Reports

Linting reports are also generated with each build and can be found in the GitHub Actions artifacts.

# Run linting locally
flake8 .
black --check .
